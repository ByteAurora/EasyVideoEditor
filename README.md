# 개발환경
- Software
  - OS: Windows 11
  - IDE: Visual Studio 2022
  - Design Tool: Qt Creator 8.0.2
- Hardware
  - CPU: Intel(R) Core(TM) i5-8400 CPU @ 2.80GHz 2.81GHz
  - RAM: DDR4 16GB
  - GPU: NVIDIA GeForce RTX 2080
- Language
  - Microsoft Visual C++ 2022
- Library
  - OpenCV 4.6.0(2022-06-12)
- Framework
  - Qt 6.4
